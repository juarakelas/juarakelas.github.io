8. Harga satu meter kain sutra adalah 4/5 kali harga satu meter kain batik. Jika Sandra membeli 3 meter kain sutra dan 5 meter kain batik seharga Rp.700.000,00 maka selisih harga per meter kedua kain tersebut adalah ....


Penjelasan

Kain sutra = s

Kain batik = b

s = 4/5b

3s + 5b = 700.000

3(4/5b) + 5b = 700.000

12/5b + 5b = 700.000

12/5b + 25/5 = 700.000

37/5b = 700.000

b = 700.000 × 5/37

b = 94.595

s = 4/5(94.595)

s = 75.676

selisih = b - s

selisih = 94.595 - 75.676 = 18.919

Jawaban : Rp.18.919

Maaf kalau salah

sutra = (4/5) batik

sutra : batik = 4n : 5n

3 sutra + 5 batik = 700.000

3 (4/5) batik + 5 batik = 700.000

(12/5) batik + (25/5) batik = 700.000

37 batik = 3.500.000

batik = 3.500.000 : 37

batik - sutra

= (3.500.000 / 37) - (4/5) batik

= (3.500.000 / 37) - (4/5)(3.500.000 /37)

= (1/5)(3.500.000 / 37)

= 700.000 / 37
