3. gaya f1, f2, f3 dan f4 bekerja pada batang abcd seperti gambar! jika massa batang diabaikan, tentukan nilai momen gaya terhadap tittk a


Maka nilai momen gaya terhadap titik A adalah - 17 Nm

Penjelasan dengan langkah langkah

Diketahui

F1  = 10 N R1 = 0 F2 = 4 N R2 = 2 meter F3 = 5 N R3 = 1 + 2 = 3 meter F4 = 4 N R4 = 1 + 2 + 3 = 6 meter

Ditanya :

Tentukan nilai momen gaya terhadap titik A

Dijawab

Jika arah momen gaya searah jarum jam = ( - ) Jika arah momen gaya berlawaman arah jarum jam = ( + ) Langkah 1 : Hitung masing masing nilai torsi

[tex]Vektor \ torsi \ = r \ x \ F \\\\F1 \ = 10 \ x\ 0 = 0 \\\\ F2 \ = ( - 2 ) \ x \ 4 \ = - 8 \ N \\\\ F3 \ = 5 \ x \ 3 \ = 15 N \\\\ F4 \ = ( - 6 ) \ x \ 4 \ = - \ 24 \ N[/tex]

Langkah 2 : Hitung torsi total

[tex]Sigma. T \ = ( - \ 8 \ N )\ + ( \ 15 \ N ) \ + \ ( \ - 24 \ N ) \\\\ Sigma. T \ = - \ 17 \ N[/tex]

Torsi materi kelas 11 bab pertama

Torsi atau biasa disebut sebagai momen gaya merupakan hasil perkalian antara gaya dengan lengan momennya. Pada lengan momennya akan diperhatikan arahnya berlawanan arah jarum jam atau searah jarum jam.

Rumus matematis dari Torsi adalah

Sigma. T = rF

ROTASI
• momen gaya

Στ di B
= F₁ sin 30° • 2 - F₃ • 2 + F₄ sin 30° • 4
= 10 √ ½ • 2 - 5 • 2 + 20 • ½ • 5
= 10 - 10 + 40
= 40 Nm ✔️

A.F1+F2-(F3+F4)
=20+20-(10+5)
=40-15
=25N ke kiri.

B.F1-(F2+F3+F4)
=20-(20+10+5)
=20-35
=-15
=15N ke kanan

C.F1-(F2+F3+F4)
=20-(20+10+5)
=20-35
=-15
=15N ke kiri

D.F1+F2+F3-(F3+F4)
=20+20+10-(10+5)
=50-15
=35N kekanan.
