12. untuk kekebalan dari penyakit ayam pada usia satu minggu harus diberi vaksin setiap 100 ekor ayam memerlukan 12 unit zat A dan 12 unit zat b di pasaran tersedia dua jenis vaksin ya itu vaksin dan vaksin n 1 bungkus vaksin b mengandung 1 unit zat A dan 3 unit zat b sedangkan vaksin mengandung 3 unit zat A dan 1 unit zat b harga per bungkus vaksin adalah rp1.000 dan vaksin rp1.500 seorang peternak mempunyai 10000 ekor ayam agar ayamnya tahan dari penyakit dan biaya yang dikeluarkan minimal dalam satu kali vaksinasi maka jumlah vaksin yang harus dibeli adalah​


Penjelasan dengan langkah-langkah:

Misalkan:  

Vaksin Jenis I = x

Vaksin Jenis II = y

Setiap 100 ekor ayam, minimal memerlukan 12 unit zat A dan 12 unit zat B. Di pasaran tersedia dua jenis vaksin yaitu vaksin Jenis I dan II. Satu bungkus vaksin I mengandung 1 unit zat A dan 3 unit zat B, sedangkan satu bungkus vaksin II mengandung 3 unit zat A dan 1 unit zat B. Diketahu juga bahwa peternak itu mempunyai 10.000 ekor ayam.

Berdasarkan pernyataan di atas, maka:

x + 3y ≥ 1200 ...(1)

3x + y ≥ 1200 ...(2)

Jika harga per bungkus vaksin I dan vaksin II masing-masing Rp. 1,000.00 dan Rp. 1,500.00, maka fungsi biayanya adalah:

f (x, y) = 1,000x + 1,500y

(1) x + 3y = 1200

x = 0 → y = 400

y = 0 → x = 1200

(x, y) = (1200, 400)

(2) 3x + y = 1200

x = 0 → y = 1200

y = 0 → x = 400

(x, y) = (400, 1200)

(3) Eliminasi dan substitusi persamaan (1) dan (2)

x + 3y = 1200 → 3x + 9y = 3600

3x + y = 1200

8y = 2400

x = 300

x = 300 → y = 300

(x, y) = (300, 300)

Fungsi biaya minimum: f (x, y) = 1,000x + 1,500y

Biaya (1) → (x, y) = (1200, 400)

Biaya (1) = 1,000x + 1,500y

Biaya (1) = 1,000 (1200) + 1,500 (400)

Biaya (1) = 1,200,000 + 600,000

Biaya (1) = 1,800,000

Biaya (2) → (x, y) = (400, 1200)

Biaya (2) = 1,000x + 1,500y

Biaya (2) = 1,000 (400) + 1,500 (1200)

Biaya (2) = 400,000 + 1,800,000

Biaya (2) = 2,200,000

Biaya (3) → (x, y) = (300, 300)

Biaya (3) = 1,000x + 1,500y

Biaya (3) = 1,000 (300) + 1,500 (300)

Biaya (3) = 300,000 + 450,000

Biaya (3) = 750,000

Jadi, biaya minimal yang harus dikeluarkan peternak tersebut dalam satu kali vaksinasi agar ayamnya tahan penyakit adalah Rp 750,000 dengan konsumsi 300 buah vaksin Jenis I dan 300 buah vaksin Jenis II.
