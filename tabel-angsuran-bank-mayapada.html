Total angsuran yang dibayarkan oleh Mr Klim dengan suku bunga 12% adalah Rp 140.000.000. Dengan nilai bunga Rp 15.000.000 dan pokok Rp 125.000.000.

Per bulan Mr Kim membayar pokok sebesar 2.083.333,33 dan bunga Rp 250.000.

Penjelasan dengan langkah-langkah:

Diketahui:

Nilai pinjaman = Rp 125.000.000

Suku bunga = 12%

Tenor pinjaman = 5 tahun atau 60 bulan

Bunga majemuk = 0,1468

Jika menggunakan asumsi suku bunga 12% maka:

Total angsuran = nilai pinjaman + (nilai pinjaman x suku bunga)Total angsuran = 125.000.000 + (125.000.000 x 12%)Total angsuran = 125.000.000 + 15.000.000Total angsuran = 140.000.000

Dari hitungan di atas diketahui bahwa jumlah bunga yang harus dibayar sebesar Rp 15.000.000. Maka

Angsuran bunga per bulan = 15.000.000 : 60 = 250.000Angsuran pokok per bulan = 125.000.000 : 60 = 2.083.333,33Sehingga total angsuran bulanan = 2.333.333,33

Jika menggunakan suku bunga majemuk 0,1468 maka:

Total angsuran = pinjaman awal x (1 + bunga[tex])^{5}[/tex]

Total angsuran = 125.000.000 x (1 + 0,1468[tex])^{5}[/tex]

Total angsuran = 125.000.000 x 1,98 = Rp 247.941.049

Sehingga:

Nilai pokok = 125.000.000 dan nilai angsuran = 122.941.049

11. pak budi meminjam uang di bank dan harus membayar angsuran selama setahun sampai hutan lunas. angsuran pertama Rp283.000,00 dan angsuran selanjutnya berkurang Rp3.000,00 dari angsuran sebelumnya. berapakah angsuran terakhir yang harus dibayar pak budi ke bank?


Diket
Angsuran pertama → a = 283.000
b = -3.000

Ditanya
U12 (karena angsuran setahun = 12 bulan)

Penyelesaian
Un = a + (n-1)b
U12 = 283.000 + (12-1)(-3.000)
U12 = 283.000 + 11(-3000)
U12 = 283.000 - 33.000
U12 = 250.000

Maka angsuran terakhir adalah Rp 250.000
