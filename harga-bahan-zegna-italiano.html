C The room is clean by somebody everyday

D Somebody is cleaned the room everyday

5. Soalno khususno:: Limitedo Editiono_____________1.) Lim x mendekati 2x(4 + 5 - 6)2.) Jika x = 5, maka [tex]\frac{ {x}^{2} + {20}^{2} }{ {x}^{4} + 400} =[/tex] adalah3.) f(x) = 25x - 5 Jika nilai x = 2maka fungsinya adalah ?Hint :maksud dari x (nomor 3) itu (25.2 - 5)Note: Alhamdulillah dapatno Malkistno Cappucino waktuno belanjano di(no) AlfamartnoSo(no) Italianoarti(no) dari(no) kata(no) hint = bantuan(no)​


Penyelesaian

Nomor 1)

Diketahui

Lim x mendekati 2x( 4 + 5 - 6 )

= ( 2.4 + 5 - 6 )

Ditanya

Hasil Limits

Jawab

( 2.4 + 5 - 6 ) = ( 8 + 5 - 6 )

= ( 13 - 6 ) = 7

Nomor 2)

Diketahui

[tex]\frac{5.5 + {20}^{2} }{5. {5}^{4} + 400 }[/tex]

x = 5

Ditanya

Hasil fungsi

Jawab

[tex] \frac{5 {x}^{2} + {20}^{2} }{ {5x}^{4} + 400 } = \frac{5.5 + {20}^{2} }{5. {5}^{4} + 400 } = \frac{25 + 400}{5.625 + 400} \\ \\ = \frac{25 + 400}{3.125 + 400} = \frac{425 \div 5}{3.525 \div 5} = \frac{85 \div 5}{141 \div 5} = \frac{17}{141} [/tex]

Nomor 3)

Diketahui

x = 2

Ditanya

Nilai Fungsi

Jawab

25.2 - 5 = 25 × 2 - 5

= 50 - 5 = 45

Kesimpulan

Nomor 1) Jawaban yang benar yaitu 7

Nomor 2) Jawaban yang benar yaitu [tex]\frac{17}{141}[/tex]

Nomor 3) Jawaban yang benar yaitu 45
