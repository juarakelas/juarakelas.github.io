2. Matematika semester 1 kelas 9 hal 49 latihan 1.4 nomor 13 beserta cara dan jawabannya


Matematika semester 1 kelas 9 hal 49 latihan 1.4 nomor 13 beserta cara dan jawabannya. Soal yang dimaksud adalah soal tentang penyederhanaab bentuk akar. Bentuk akar merupakan salah satu bilangan irasional yaitu bilangan yang tidak dapat diubah menjadi bentuk a/b dengan a, b bilangan bulat dan a ≠ 0. Operasi hitung pada bentuk akar:

Penjumlahan dan pengurangan

a√b + c√b = (a + c) √b a√b + c√d = a√b + c√d √a + √a = 2√a

Perkalian bentuk akar

√a × √a = a √a × √b = √(a.b) a × √b = a√b a√b × c√d = a.c√(b.d)

Pembahasan

a. √112

= √(16 × 7)

= √16 × √7

= 4√7

b. √216

= √(36 × 6)

= √36 × √6

= 6√6

c. √605

= √(121 × 5)

= √121 × √5

= 11√5

d. √800

= √(400 × 2)

= √400 × √2

= 20√2

e. √5.000

= √(2.500 × 2)

= √2.500 × √2

= 50√2

f. √0,000121

= √(121 × 0,000001)

= √121 × √0,000001

= 11 × 0,001

= 0,011

g. √0,00000324

= √(324 × 0,00000001)

= √324 × √0,00000001

= 18 × 0,0001

= 0,0018

h. 9√2 + √72 – √578

= 9√2 + √(36 × 2) – √(289 × 2)

= 9√2 + √36 × √2 – √289 × √2

= 9√2 + 6√2 – 17√2

= (9 + 6 – 17)√2

= –2√2

i. 7√3 + √48 – √768

= 7√3 + √(16 × 3) – √(256 × 3)

= 7√3 + √16 × √3 – √256 × √3

= 7√3 + 4√3 – 16√3

= (7 + 4 – 16) √3

= –5√3

j. 9√5 – √125 + √720

= 9√5 – √(25 × 5) + √(144 × 5)

= 9√5 – √25 × √5 + √144 × √5

= 9√5 – 5√5 + 12√5

= (9 – 5 + 12) √5

= 16√5

4. jawaban tabel 1.3 dan 1.4 matematika hal 25 kelas 7 ​


Hasil dari a x (b + c) dan (a x b) + (a x c) adalah sama.

Hasil dari a x (b - c) dan (a x b) - (a x c) adalah sama.

Sehingga dapat disimpulkan bahwa kolom 6 dan 9 memenuhi sifat distributif.

Pengecekan sifat distributif pada perkalian dan penjumlahan.

Tabel 1.3

1. a = 1, b = 5, c = 4

2. a = -2, b = 6, c = -3

3. a = 3, b = -7, c = 2

4. a = -4, b = -8, c = -1

Tabel 1.4

1. a = 1, b = 5, c = 4

2. a = -2, b = 6, c = -3

3. a = 3, b = -7, c = 2

4. a = -4, b = -8, c = -1

Penjelasan dengan langkah-langkah:

Diketahui:

Tabel 1.3

1. a = 1, b = 5, c = 4

2. a = -2, b = 6, c = -3

3. a = 3, b = -7, c = 2

4. a = -4, b = -8, c = -1

Tabel 1.4

1. a = 1, b = 5, c = 4

2. a = -2, b = 6, c = -3

3. a = 3, b = -7, c = 2

4. a = -4, b = -8, c = -1

Ditanya:

Mengamati hasil dari a x (b + c) dan (a x b) + (a x c).

Mengamati hasil dari a x (b - c) dan (a x b) - (a x c).

Jawab:

Tabel 1.3

1. a = 1, b = 5, c = 4

⇔ a x (b + c)

= 1 x (5 + 4)

= 1 x 9

= 9

⇔ (a x b) + (a x c)

= (1 x 5) + (1 x 4)

= 5 + 4

= 9

2. a = -2, b = 6, c = -3

⇔ a x (b + c)

= -2 x (6 + (-3))

= -2 x 3

= -6

⇔ (a x b) + (a x c)

= (-2 x 6) + (-2 x (-3))

= -12 + 6

= -6

3. a = 3, b = -7, c = 2

⇔ a x (b + c)

= 3 x (-7 + 2)

= 3 x (-5)

= -15

⇔ (a x b) + (a x c)

= (3 x (-7)) + (3 x 2)

= -21 + 6

= -15

4. a = -4, b = -8, c = -1

⇔ a x (b + c)

= -4 x (-8 + (-1))

= -4 x (-9)

= 36

⇔ (a x b) + (a x c)

= (-4 x (-8)) + (-4 x (-1))

= 32 + 4

= 36

Tabel 1.4

1. a = 1, b = 5, c = 4

⇔ a x (b - c)

= 1 x (5 - 4)

= 1 x 1

= 1

⇔ (a x b) - (a x c)

= (1 x 5) - (1 x 4)

= 5 - 4

= 1

2. a = -2, b = 6, c = -3

⇔ a x (b - c)

= -2 x (6 - (-3))

= -2 x 9

= -18

⇔ (a x b) - (a x c)

= (-2 x 6) - (-2 x (-3))

= -12 - 6

= -18

3. a = 3, b = -7, c = 2

⇔ a x (b - c)

= 3 x (-7 - 2)

= 3 x (-9)

= -27

⇔ (a x b) - (a x c)

= (3 x (-7)) - (3 x 2)

= -21 - 6

= -27

4. a = -4, b = -8, c = -1

⇔ a x (b - c)

= -4 x (-8 - (-1))

= -4 x (-7)

= 28

⇔ (a x b) - (a x c)

= (-4 x (-8)) - (-4 x (-1))

= 32 - 4

= 28

Ayo kita berlatih 1.4 nomor 4 matematika kelas 8

Tentukan dua suku berikutnya dari barisan bilangan berikut berdasarkan pola bilangan sebelumnya.  

A. 2, 3, 4, 6, 6, 12, 8 , ... , ...B. 3, 7, 11, 18, ... , ...C. 1, 2, 5, 14, .... , .....D. 81, 80, 27, 40, 9, .... , ....E. 1, 3, 4, 9, 9, 27, 16, .... , ....

Pola bilangan adalah rangkaian dari beberapa angka yang membentuk pola yang tretentu.

Pembahasan

A.  2 , 3 , 4 , 6 , 6 , 12 , 8 ,       24 , 10

Pola I = 2 , 4 , 6 , 8, 10, ...

Berpola + 2 dari bilangan sebelumnya

suku berikutnya → 8 + 2 = 10

 Pola II = 3 , 6 , 12 , ...

Berpola × 2 dari bilangan sebelumya

suku berikutnya → 12 × 2 = 24

Jadi dua suku berikutnya adalah 24 dan 10

B.  3 , 7 , 11 , 18 ,       19 , 29

Pola I = 3 , 11 , 19, ...

Berpola + 8 dari bilangan sebelumnya

suku berikutnya → 11 + 8 = 19

Pola II = 7 , 18 , 29, ....

Berpola + 11 dari bilangan sebelumya

suku berikutnya → 18 + 11 = 29

Jadi dua suku berikutnya adalah 19 dan 29

C.  1  ,  2  ,  5  ,  14  ,       41, 122

     +1   +3    +9

         ×3   ×3

suku ke-5 = 3 × 9 + 14 = 41

suku ke-6 = 3 × 3 × 9 + 41 = 122

Jadi dua suku berikutnya adalah 41 dan 122

D. 81 , 80,  27 , 40 , 9 ,      20 , 3

Pola I = 81 , 27 , 9, ....

Berpola dibagi 3 dari bilangan sebelumnya

suku berikutnya → 9 : 3 = 3

Pola II = 80 , 40 , ....

Berpola dibagi 2 dari bilangan sebelumya

suku berikutnya → 40 : 2 = 20

Jadi dua suku berikutnya adalah 20 dan 3

E. 1 , 3 , 4 , 9 , 9 , 27 , 16 ,      81 , 25

Pola I = 1 , 4 , 9 , 16, ....

Berpola bilangan kuadrat 

suku berikutnya → 5² = 25

Pola II = 3 , 9 , 27, ...

Berpola dikali 3 dari bilangan sebelumya

suku berikutnya → 27 × 3 = 81

Jadi dua suku berikutnya adalah 81 dan 25

Pada bilangan 100100100100100……

Setelah angka  0 kembali ke angka 1 kemudian 0 dan seterusnya artinya pada bilangan ini membentuk pola berulang. Sehingga untuk menjawab pertanyaanya kali bilangan tersebut akan dibagi menjadi kelompok angka berulang, kelompok angka itu adalah 100, kelompok ini berulang terus sampai tak terhingga, sehingga

100  |  100  |  100  |  100  |  ……

Karena pada bilangan di atas setelah urutan ke 3, terjadi perulangan maka dengan membagi bilangan tersebut dengan 3 kemudian diperoleh sisa hasil bagi dari pembagian tersebut. Contohnya untuk mencari angka ke 8, dengan membagi 8 dengan 3 hasilnya adalah 2 dan sisa hasil baginya 2, maka angka yang dicari adalah angka urutan ke 2 pada kelompok yaitu 0.

Sifat Bentuk Akar

[tex]\sqrt{a^{2} } = a[/tex][tex]\sqrt{a} = a^{\frac{1}{2}}[/tex][tex]\sqrt{a} \times \sqrt{b} = \sqrt{ab}[/tex][tex]x \sqrt{a} + y\sqrt{a} = (x + y) \sqrt{a}[/tex][tex]x \sqrt{a} - y\sqrt{a} = (x - y) \sqrt{a}[/tex][tex]\frac{\sqrt{a} }{\sqrt{b}} = \sqrt{\frac{a}{b}}~~ atau~ \sqrt{a} : \sqrt{b} = \sqrt{a : b}~~ , b\neq 0[/tex]

Untuk sifat-sifat dalam menyedehanakan bentuk akar bisa dilihat pada lampiran II

Pembahasan

a. √112

         = 6√6

          = √121 × √5

          = 11√5

d. √800

√800 = [tex]\sqrt{400 \times 2}[/tex]

          = √400 × √2

          = 20√2

e. √5.000

√5.000 = [tex]\sqrt{2500 \times 2}[/tex]

             = √2500 × √2

             = 50√2

f. √0,000121

                  = √121 : √1.000.000

                  = 11 : 1.000

                  = 0,011

g. √0,00000324

√0,00000324 = [tex]\sqrt{324 : 100.000.000}[/tex]

                        = √324 : √100.000.000

                        = 18 : 10.000

                        = 0,0018

h. 9√2 + √72 - √578

9√2 + √72 - √578 = 9√2 + [tex]\sqrt{36 \times 2}[/tex] - [tex]\sqrt{289 \times 2}[/tex]

                               = 9√2 + 6√2 - 17√2

                               = 15√2 - 17√2

                               = -2√2

= 7√3 + [tex]\sqrt{16 \times 3}[/tex] - [tex]\sqrt{256 \times 3}[/tex]

= 7√3 + 4√3 - 16√3

= 11√3 - 16√3

= -5√3

= 16√5
