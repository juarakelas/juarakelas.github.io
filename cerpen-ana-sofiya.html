Jawab:

Umur Sofiya adalah 12 Tahun

Penjelasan dengan langkah-langkah:

Umur Sofiya = a

Umur Ayah = b

1. Umur Sofiya lebih muda 32 tahun dari Ayah :

a = b - 32

2. Jumlah umur Sofiya dan Ayah adalah 56:

a + b = 56

3. dari info nomor 2, umur Ayah (b) berarti :

a + b = 56

b = 56 - a

4. dari info nomor 1, umur Sofiya (a) berarti :

a = b - 32

ganti huruf b dengan info nomor 3 :

a = (56 - a) -32

a = 56 - 32 - a

2a = 24

a = 12
