1. Nilai dari 1 + 2.2 + 3.2² + 4.2³ + ... + 2018.2²⁰¹⁷ adalah...


1 + 2.2 + 3.2^2 + 4.2^3 + ... + 2018.2^(2017) = (n).2^(n - 1) [n = 1 .. 2018]

= (n).(2^n)/2


= n.2^n + 2^n [n = 0 .. 2017]

Untuk n.2^n
1.2^1 = 2 = 2 + 0.2^2
1.2^1 + 2.2^2 = 2 + 8 = 2 + 1.2^3
1.2^1 + 2.2^2 + 3.2^3 = 2 + 8 + 24 = 2 + 32 = 2 + 2.2^4
dst.
maka,
Sum n.2^n = 2 + (n - 1).2^(n + 1)

Maka,
(n).(2^n)/2 [n = 1 .. 2018] = (2 + (2018 - 1).2^(2018 + 1))/2
= 1 + 2017.2^(2018 + 1 - 1)
= 1 + 2017.2^(2018)

2. nilai 1+2.2+3.2²+4.2³+...+2018.2 pangkat 2017 =...


1+2.2+3.2²+4.2³+...+2018.2²⁰¹⁷

= 1.2⁰+2.2¹+3.2²+4.2³+...+2018.2²⁰¹⁷

pola bilangannya

= n.2ⁿ⁻¹

= n(2ⁿ.2⁻¹)

= n(2ⁿ) : 2

n(2ⁿ) : 2 adalah bentuk pecahan sehingga bila dijumlahkan, cukup dijumlahkan pembilangnya, penyebutnya tetap, yaitu 2.

pola jumlah bilangan untuk n(2ⁿ)

S₁ => n = 1 => 1.2¹= 2

S₂ => n = 2 => 1.2¹+ 2.2² = 10

S₃ => n = 3 => 1.2¹+2.2²+3.2³ = 34

S₄ => n = 4 => 34 +4.2⁴ = 34 + 64 = 98

S₅ => n = 5 => 98 + 5.2⁵ = 98 + 160 = 258

S₆ => n = 6 => 258 + 6.2⁶ = 258+384 = 642

pola jumlah bilangan dari

......... 2, 10, 34, 98, 258, 642,

.........u₁, u₂, u₃, u₄, u₅, u₆

= (2+ 0.2²), (2 + 1.2³),(2 + 2.2⁴),(2 + 3.2⁵), ...

sehingga jumlah bilangan n suku berpola n(2ⁿ) adalah 2 + (n - 1).2ⁿ⁺¹.

Karena suku terakhir n = 2018 maka jumlahnya adalah

= 2 + (2018-1).2²⁰¹⁸⁺¹

= 2 + 2017.2²⁰¹⁹

maka

untuk bilangan berpola, n(2ⁿ):2 yang banyak sukunya 2018 buah, nilainya adalah:

= (2 + 2017.2²⁰¹⁹) : 2

= ((2:2) + (2017.2²⁰¹⁹ : 2))

= 1 + 2017.2²⁰¹⁹⁻¹

= 1 + 2017.2²⁰¹⁸

Jadi nilai dari

1+2.2+3.2²+4.2³+...+2018.2²⁰¹⁷

= 1 + 2017.2²⁰¹⁸

4. Soal iseng", di jawab dengan menggunakan cara dan langkah yang benar yaa. Nilai dari [tex]1+ 2.2+3.2^2+4.2^3+....+2018.2^{2017} [/tex] sama dengan . . .


Misalkan :
A   = 1 +    2 . 2    + 3 . 2^(2) + 4 . 2^(3) + ... + 2018 . 2^(2017)
2A = 2 + 2 . 2^(2) + 3 . 2^(3) + 4 . 2^(4) + ... + 2018 . 2^(2018)

A   = 1 + 2.2 + 3.2^(2) + 4.2^(3) + ... + 2018.2^(2017)
2A =          2 + 2.2^(2) + 3.2^(3) + ... + 2017.2^(2017) + 2018.2^(2018)    _
-A  = 1 +    2 +   2^(2)  +   2^(3)  + ... +     2^(2017)      - 2018.2^(2018)

-A  = 1 + 2 + 2^(2) + 2^(3) + ... + 2^(2017) - 2018.2^(2018)
A   = 2018.2^(2018) - (1 + 2 + 2^(2) + 2^(3) + ... + 2^(2017))
A   = 2018.2^(2018) - (2^(0) . (2^(2018) - 1)
                                                2-1
A   = 2018 . 2^(2018) - (2^(2018) - 1)
A   = 2018 . 2^(2018) - 2^(2018) + 1
A   = 2^(2018) (2018 - 1) + 1
A   = 2^(2018) . 2017 + 1

Jadi, nilai dari 1 + 2 . 2 + 3 . 2^(2) + 4 . 2^(3) + ... + 2018 . 2^(2017) sama dengan 2^(2018) . 2017 + 1
