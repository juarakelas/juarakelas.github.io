8. Bagaimana cara memindahkan folder atau alamat website


1) Klik tombol Favorites di toolbar,maka kotak task pane Favorites akan ditampilkan.
2) Pilih dan klik tombol Organize ,maka kotak dialog Organize Favorites akan ditampilkan.
3) Pilih folder atau alamat website yang ingin dipindahkan,kemudian pilih dan klik tombol Move to folder maka kotak dialog Browse for Folder akan ditampilkan.
4) Pilih folder baru di mana folder atau alamat website dipindahkan.
5) Klik tombol OK untuk melakukan proses pemindahan data ke tempat baru.

= (5×10) - 5 thn

= 50 - 5 thn

= 45 thn

= 45 thn - (1×5) thn

= 40 thn

Selisih usia Ayah Fatma dan Ibu Fatma

= 45-40 = 5 thn
