Jawab:

1.) Grafik fungsi kuadrat memotong sumbu x di titik ( -1, 0 ) dan ( 5, 0 ) serta memotong sumbu y di titik ( 0, 5 )

Maka Persamaan grafik fungsi kuadratnya adalah :

y = -x² + 4x + 5

Penjelasan dengan langkah-langkah:

Persamaan grafik fungsi kuadrat dapat dicari jika diketahui syarat sbb :

1. Grafik memotong sumbu x di ( x₁, 0 ) dan ( x₂, 0 ) serta melalui titik sembarang ( x₃, y₃ ) pada grafik, maka persamaannya adalah y = a( x - x₁ )( x - x₂).

2. Grafik mempunyai titik balik ( xₙ, yₙ ) serta melalui titik sembarang ( x₁, y₁ ) pada grafik, maka persamaannya adalah y = a(x - xₙ)² + yₙ

3. Grafik melalui tiga buah titik yaitu ( x₁, y₁ ), ( x₂, y₂ ) dan ( x₃, y₃), maka persamaannya adalah y = ax² + bx + c.

Pada soal tersebut menggunakan syarat no. 1

Memotong sumbu x di titik (  ( -1, 0 ) dan ( 5, 0 ) serta memotong sumbu y di titik ( 0, 5 )

Rumusnya :

y = a( x - x₁ )( x - x₂)

Kita cari nilai a terlebih dahulu

y = a( x - x₁ )( x - x₂)

5 = a ( 0 - (-1) ) ( 0 - 5 )

5 = a (1) (-5)

5 = -5a

a = -1

Kemudian kita masukkan nilai a ke fungsi persamaan

y = a( x - x₁ )( x - x₂)

y = -1 ( x - (-1) ) ( x - 5 )

y = -1 ( x + 1 ) ( x - 5 )

y = -1 ( x² - 5x + x  - 5 )

y = -1 ( x² - 4x - 5 )

y = -x² + 4x + 5

2.)  Pembahasan :

Suatu fungsi dari himpunan A ke himpunan B adalah suatu relasi khusus sedemikian hingga setiap anggota himpunan A dipasangkan dengan tepat satu anggota himpunan B.

Himpunan A dinamakan domain atau daerah asal dan himpunan B dinamakan kodomain atau daerah kawan.

Jika fungsi f memetakan setiap x ∈ A dengan tepat ke satu anggota y ∈ B, maka f : x → y.

Peta dari x ∈ A oleh fungsi f sering dinyatakan sebagai f(x) dan bentuk f(x) dinamakan rumus fungsi f.

Himpunan y ∈ B yang merupakan peta dari x ∈ A dinamakan range atau daerah hasil.

Misalkan fungsi f dinyatakan dengan f : x → ax + b dengan a dan b merupakan konstanta dan x merupakan variabel, maka rumus fungsinya adalah f(x) = ax + b.

Jika nilai variabel x = m, maka nilai f(m) = am + b.

Dengan demikian, kita dapat menentukan bentuk fungsi f, bila diketahui nilai-nilai fungsinya.

Selanjutnya, nilai konstanta a dan b ditentukan berdasarkan nilai-nilai fungsi yang diketahui.

4. a. Ruang sampel S = {(A, 2), (A, 3), (A, 8), (A, 9), (B, 2),  (B, 3), (B, 8), (B, 9), (C, 2), (C, 3), (C, 8), (C, 9), (D, 2), (D, 3), (D, 8), (D, 9)}.

Penjelasan dengan langkah-langkah:

Jika S adalah himpunan ruang sampel dari percobaan dengan setiap anggota dari S memiliki kesempatan muncul yang sama.

A adalah himpunan kejadian dengan A ⊂ S.

Peluang kejadian A adalah

P(A) = [tex]\frac{n(A)}{n(S)}[/tex]

dengan n(A) merupakan banyaknya anggota dalam himpunan kejadian A dan n(S) merupakan banyaknya anggota dalam himpunan ruang sampel S.

Dua kejadian A dan B dalam ruang sampel S dinamakan saling lepas atau saling asing bila dalam percobaan yang menghasilkan ruang sampel S tersebut, kedua kejadian A dan B tersebut tidak mungkin terjadi secara bersamaan.

P(A ∪ B) = P(A) + P(B) - P(A ∩ B)

dan P(A ∩ B) = ∅.

Diketahui:

Kotak I berisi 5 bola A, B, C, D, dan E.

Kotak II berisi 4 bola 2, 3, 8, dan 9.

Setiap kotak diambil sebuah bola secara acak.

Ditanyakan:

a. Ruang sampel

b. Peluang huruf vokal atau bilangan prima

c. Peluang selain bilangan kuadrat.

Jawab:

a. S adalah ruang sampel.

S = {(A, 2), (A, 3), (A, 8), (A, 9), (B, 2),  (B, 3), (B, 8), (B, 9), (C, 2), (C, 3), (C, 8), (C, 9), (D, 2), (D, 3), (D, 8), (D, 9)}

n(S) = 20

Banyaknya anggota S adalah 20.

b. V adalah himpunan dari huruf vokal.

V = {A, E}

n(V) = 2.

Banyaknya anggota V adalah 2.

W adalah himpunan dari bilangan prima.

W = {2, 3}

n(W) = 2.

Banyaknya anggota W adalah 2.

Peluang huruf vokal atau bilangan prima:

P(V ∪ W) = [tex]\frac{n(V)}{n(S)}[/tex] + [tex]\frac{n(W)}{n(S)}[/tex]

⇔ P(V ∪ W) = [tex]\frac{2}{20}[/tex] + [tex]\frac{2}{20}[/tex]

⇔ P(V ∪ W) = [tex]\frac{4}{20}[/tex]

⇔ P(V ∪ W) = [tex]\frac{1}{5}[/tex]

Jadi, peluang huruf vokal atau bilangan prima adalah [tex]\frac{1}{5}[/tex].

c. K adalah himpunan dari selain bilangan kuadrat.

K = {2, 3, 8}

n(K) = 3

Banyaknya anggota K adalah 3.

Peluang selain bilangan kuadrat:

P(K) = [tex]\frac{n(K)}{n(S)}[/tex]

⇔ P(K) = [tex]\frac{3}{20}[/tex]

Jadi, peluang selain bilangan kuadrat adalah [tex]\frac{3}{20}[/tex].

Pembahasan

Pada pernyataan kedua, tertulis "Istambul". Kemungkinan terdapat kesalahan penulisan. Kata yang seharusnya tertulis adalah "Istanbul", yang merupakan ibukota dari negara Turki. Untuk pembahasan soal ini, digunakan penulisan "Istanbul".

Ada empat jenis pernyataan majemuk, yaitu:

Disjungsi (atau, ditandai dengan ∨)Konjungsi (dan, ditandai dengan ∧)Implikasi (jika-maka, ditandai dengan ⇒)Biimplikasi (jika dan hanya jika, ditandai dengan ⇔)

Pernyataan disjungsi hanya salah jika kedua pernyataan di dalamnya bernilai salah. Pernyataan konjungsi hanya benar jika kedua pernyataan di dalamnya bernilai benar. Pernyataan implikasi hanya salah jika pernyataan pertama bernilai benar dan pernyataan kedua bernilai salah. Pernyataan biimplikasi hanya benar jika kedua pernyataan di dalamnya bernilai sama (sama-sama benar atau sama-sama salah).

Berikut pembahasan nilai kebenaran dari masing-masing pernyataan:

5 merupakan bilangan prima, tetapi bukan bilangan genap. 5 merupakan bilangan ganjil. Karena ada sebuah pernyataan yang benar, maka pernyataan disjungsi ini bernilai benar.Tokyo merupakan ibukota Jepang dan Istanbul merupakan ibukota Turki. Karena kedua pernyataannya benar, maka pernyataan implikasi ini bernilai benar.India terletak di Benua Asia (tepatnya di Asia Selatan), tetapi Amerika Serikat bukan terletak di Benua Asia, melainkan di Benua Amerika (tepatnya di Amerika Utara). Karena ada sebuah pernyataan yang salah, maka pernyataan konjungsi ini bernilai salah.16 merupakan bilangan kuadrat sempurna, tetapi 19 tidak habis dibagi 2. Ketika dibagi 2, 19 akan bersisa 1 (tidak habis dibagi). Karena kedua pernyataan bernilai tidak sama, maka pernyataan biimplikasi ini bernilai salah.

Tabel kebenaran dapat dibuat dengan mendaftarkan semua kemungkinan nilai kebenaran dari setiap pernyataan. Ingat bahwa hanya ada dua nilai kebenaran, yaitu benar (B) dan salah (S). Lalu, setiap kemungkinan dioperasikan dengan masing-masing tanda pernyataan majemuk, sesuai dengan aturan yang telah disebutkan sebelumnya.
