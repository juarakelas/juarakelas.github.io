3. Tinggi rata-rata 15 orang wanita sama dengan 160 cm. Jika digabung dengan pria yang tinggi rata-ratanya sama dengan 170 cm, maka tinggi rata-ratanya menjadi sama dengan 166,25. Jumlah pria yang digabung sama dengan ...


Jumlah pria yang digabung sama dengan 25 Orang

Penjelasan dengan langkah-langkah:

Diketahui :

Tinggi rata-rata wanita = 160 cm

Banyak wanita              = 15 orang

Tinggi rata-rata pria      = 170 cm

Rata-rata Gabungan      = 166,25

Ditanya :

Banyak pria ?

Jawab:

(x ) ̅gabungan =[tex]\frac{n_1 (x ̅_1 ) ̅+ n_2 x ̅_2}{n_1+n_2}[/tex]

Keterangan:

n 1= Jumlah anggota kelompok 1

n 2= Jumlah anggota kelompok 2

x 1 = Nilai rata-rata kelompok 1

x 2 = Nilai rata-rata kelompok 2

166,25 =[tex]\frac{(15x160)+(170xB)}{15+B}[/tex]

166,25 x (15+B) = (15 x 160) + (170 x B)

2493,75 + 166,25B = 2400 + 170 B

2493,75-2400  = 170B - 166,25B

93,75       = 3,75B

B  = [tex]\frac{93,75}{3,75}[/tex]

B = 25

Jadi Jumlah pria yang digabung sama dengan 25 Orang

4. Tinggi rata-rata murid wanita adalah 162 cm, sedangkan tinggi rata-rata yang pria 170 cm. Jika rata-rata tinggi keseluruhan 165 cm. Perbandingan banyak murid pria dan wanita. Adalah…….


x̄gab = x̄1 • n1 + x̄2 • n2 per n1 + n2
165 = 162 • n1 + 170 • n2 per n1 +n2
165n1 + 165n2 = 162n1 + 170n2
165n1 - 162n1 = 170n2 - 165n2
3n1 = 5n2
n2 per n1 = 3 per 5
jadi perbandingan banyak murid pria dan wanita adalah 3:5
