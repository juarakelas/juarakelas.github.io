Penjelasan:

Diket: P = -6Q + 36

TR = P.Q

TR maksimal dapat diperoleh jika MR = 0. Jadi kita perlu mencari MR terlebih dahulu.

a. TR = P.Q

= (-6Q + 36)Q

= -6Q² + 36Q

b. TR’ = turunan dari TR

TR = -6Q + 36 = MR

MR = 0

-6Q + 36 = 0

-6Q = -36

Q = 6

Jadi,Q maksimumnya adalah 6

selanjutnya kita akan mencari TR maksimum,dimana TR maksimum dapat diperoleh jika Q = 6. Nah ini kita substitusi kan ke dalam fungsi TR :

TR = -6Q² + 36Q

TR = -6(6)²+36(6)

= -6(36) + 126

= -126 + 126

TR = 0

12. P=20-Q tentukan fungsi penerimaan total(TR)


Fungsi penerimaan total (TR) jika P = 20 – Q adalah TR = 20Q – Q². Jika kita akan mencari penerimaan maksimumnya, maka jawabannya adalah 100 saat Q = 10.

Penjelasan dengan langkah-langkah

Rumus fungsi penerimaan total (TR)

TR = P × Q

Keterangan

P = fungsi penerimaanQ = banyaknya unit barang

Penerimaan akan bernilai maksimum jika

TR’ = 0 (turunan pertamanya sama dengan nol)

Diketahui

Fungsi permintaan: P = 20 – Q.

Ditanyakan

Tentukan fungsi penerimaan total (TR)!

Jawab

Langkah 1

Fungsi penerimaan total.

TR = P × Q

    = (20 – Q) × Q

    = 20Q – Q²

Langkah 2

Jika kita akan mencari penerimaan maksimumnya, maka kita cari turunan pertama dari TR dan hasilnya sama dengan nol.

TR = 20Q – Q²

TR’ = 20 – 2Q

   0 = 20 – 2Q

2Q = 20

  Q = [tex]\frac{20}{2}[/tex]

  Q = 10

Langkah 3

Penerimaan maksimum yang diperoleh terjadi saat Q = 10, yaitu:

TR = 20Q – Q²

    = 20(10) – 10²

    = 200 – 100

    = 100

15. Diketahui fungsi penerimaan TR=50Q, fungsi biaya TC=120+20Q. Berapa BEP!​


Pendahuluan:

Break even point atau BEP merupakan kondisi ketika pendapatan sebuah perusahaan sama dengan modal yang keluarkan. Akuntansi menyebut BEP sebagai titik impas, karena perusahaan tidak mengalami untung maupun rugi.

Diketahui:

TR = 50Q

TC = 120 + 20 Q

Jawaban:

Untuk menghitung titik impas digunakan rumus titik impas = break-even point atau TR = TC, maka dengan menggunakan rumus di atas, titik impas nya adalah:

[tex]TR=TC\\50Q=120+20Q\\30Q=120\\Q= 4[/tex]

Jadi, titik impas/BEP-nya sebesar 4 unit.
