Jawab: pola tersebut merupakan barisan aritmatika jadi :

Suku awal=a suku awal segi lima tersebut adalah 5

Suku ke 2 yaitu U2 rumusnya yaitu a+b=8

Suku ke 3 yaitu U3 rumusnya yaitu

a+2b=11

Apakah itu b?

b adalah selisih dari bilangan sebelumnya atau biasa disebut dengan beda

Berapakah bedanya?

b =U3-U2=11-8=3

Maka b nya adalah 3

Mencari suku ke 6?

Maka n nya=6

Rumusnya a+(n-1).b

5+(6-1).3=5+(5).3=5+15=18

Maka banyak nya korek api yang di butuhkan untuk urutan ke 6 adalah 18 buah

Penjelasan dengan langkah-langkah:

Banyaknya batang korek api yang diperlukan pada urutan ke - 6 dari barisan segi lima beraturan tersebut adalah 25 batang korek api

Pembahasan

Barisan aritmatika adalah barisan bilangan dimana diantara dua suku yang berurutan mempunyai selisih (beda) yang konstan (tetap).

Rumus Beda

b = Uₙ - Uₙ₋₁

Rumus Suku ke - n

Uₙ = a + (n - 1)b

..

Dari penjelasan tersebut mari selesaikan soal berikut.

Diketahui :

Perhatikan Gambar Tersebut

• Suku ke 1 terdiri dari 5 batang korek api

• Suku ke 2 terdiri dari 9 batang korek api

• Suku ke 3 terdiri dari 13 batang korek api

Maka, dari gambar tersebut dapat dibuat sebuah barisan berikut.

5, 9, 13, ...

Ditanya :

Banyak korek api pada urutan ke - 6 (U₆) ?

Jawab :

5, 9, 13, ...

Suku pertama (a) = 5

Beda (b) = U₂ - U₁ = 9 - 5 = 4

Suku ke - 6

Uₙ = a + (n - 1)b

U₆ = 5 + (6 - 1)(4)

U₆ = 5 + (5)(4)

U₆ = 5 + 20

U₆ = 25

..

Jadi, Banyak korek api yang diperlukan pada urutan ke - 6 dari barisan segi lima beraturan tersebut adalah 25 batang korek api.

15. Pada gambar berilut berapakah bangun segi lima pada urutan ke 25 (Un) dengan membuat rumusan polanya terlebih dahulu*pakai cara​


Jawab:

Semoga dapat terbantu ya :)

Diketahui:

Urutan ke-1 (U1) = 1 segi lima

Urutan ke-2 (U2) = 4 segi lima

Urutan ke-3 (U3) = 7 segi lima

Ditanya:

Urutan ke-25 (U25) = ? segi lima

Dijawab:

Rumus suku ke-n barisan aritmatika

  Un = a + b(n-1)

   Un = suku ke -n

   a = suku ke -1  = 1

   b = beda (selisih suku ke n dan n-1), b = suku ke-2 – suku ke-1 = 4 – 1 = 3

 Un = a + b(n-1) = 1 + 3(n-1) = 1 + 3n -3 = 3n-2

Urutan ke-25 (U25) = 3(25)-2= 75 -2 = 73 segi lima
