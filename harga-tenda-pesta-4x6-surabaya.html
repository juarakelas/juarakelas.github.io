6. sebuah tenda berbentuk kerucut akan dibuat dari kain Jika harga kain untuk alas Rp14.000 per meter dan 21000 meter untuk dinding tenda harga tenda jika r = 1 m dan S = 3 m adalah​


Jawab:

Penjelasan dengan langkah-langkah:

Bangun ruang kerucut adalah bangun ruang yang memiliki alas berbentuk lingkaran dan selimut yang berbentuk irisan dari lingkaran.

L permukaan = luas alas + luas selimut

Luas selimut = x r x s

Luas alas = x r², dimana :

= jika r kelipatan 7

= 3,14 jika r bukan kelipatan 7

r = jari - jari lingkaran

Pembahasan

L permukaan kerucut = luas alas + luas selimut

Luas selimut kerucut = x r x s

Luas selimut kerucut = 3,14 x 1 m x 3 m

Luas selimut kerucut = 9,42 m²

Jika harga kain untuk dinding tenda adalah Rp 21.000,00 per m², maka harga kain untuk dinding tenda adalah :

= 9,42 m² x Rp 21.000,00

= Rp 197.820,00

Luas alas tenda = x r²

Luas alas tenda = 3,14 x 1 m x 1 m

Luas alas tenda = 3,14 m²

Jika harga kain untuk alas tenda adalah Rp 14.000,00 per m², maka harga kain untuk alas tenda adalah :

= 3,14 m² x Rp 14.000,00

= Rp 43.960,00

Jadi, total harga untuk membeli kain tenda adalah :

= harga kain untuk dinding tenda + harga kain untuk alas tenda

= Rp 197.820,00 + Rp 43.960,00

= Rp 241.780,00
